### Pins:
| Arduino | TFT Display |
|---------|-------------|
| 3v3     | LED         |
| 13      | SCK         |
| 11      | SDA         |
| 3       | AO          |
| 2       | RESET       |
| 4       | CS          |
| GND     | GND         |
| 5v      | VCC         |
